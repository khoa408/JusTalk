python 3<br/>
pip3<br/>

Google Translate client library: `pip3 install --upgrade google-cloud-translate`<br/>
Google Text-to-Speech client library: `pip3 install --upgrade google-cloud-texttospeech` (not yet implemented)<br/>
Pyaudio<br/>
Tkinter<br/>
